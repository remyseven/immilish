# Immilish 🌍

**AI-Powered Language Learning App — Learn 107 Languages**

Created by Tyler · TYLM LLC

---

## About

Immilish is an iOS language learning app that teaches 107 languages through 10 progressive levels — from basic words all the way to writing full essays. Every answer gets real-time AI coaching from the Immilish AI Coach.

### Features
- 🌍 107 languages across 6 world regions + sacred languages
- 🤖 AI coaching after every answer
- 💬 Conversation Buddy with drag-and-drop word tiles
- 📖 Tap-to-define paragraph reader (Filipino Level 5)
- 🏆 10 levels: Basic Words → Full SCR Essays
- 📱 Age 4+ · No ads · No accounts required
